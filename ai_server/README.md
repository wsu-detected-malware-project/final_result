# ai_server
## 소개
  AI 기반 악성코드 판단 서버입니다.
  중계서버에서 받아온 데이터를 AI 기반으로 악성코드를 판단하여 송신하는 서버입니다.
## 설치방법
### 사전 요구 사항
- Visual Studio Code
- Python 또는 Anaconda
