# db_server
## 소개
  DB에 데이터를 삽입하는 코드 모음집 입니다.
  DB테이블에 전처리된 PE 데이터를 삽입하는 코드와 해시화 하는 코드, 전체 출력하는 코드가 들어있습니다.
## 설치방법
### 사전 요구 사항
- Visual Studio Code
- Python 또는 Anaconda
- Microsoft SQL Server
